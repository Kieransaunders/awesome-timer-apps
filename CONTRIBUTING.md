# Contributing to Awesome Timer Apps

Thanks for taking the time to contribute!

## Guidelines

- The app must be primarily a timer or time-tracking tool
- It must be actively maintained (updated within the last 12 months)
- It must be publicly available (free, freemium, or paid)
- Add a short, factual description — no marketing fluff
- Place it in the most relevant category
- Keep alphabetical order within each category where possible

## How to Submit

1. Fork this repository
2. Add your entry to the relevant section in `README.md`
3. Submit a pull request with a brief reason for inclusion

## Format

```
- **[App Name](https://url.com)** — One sentence description. Platforms and pricing note.
```

## What won't be accepted

- Abandoned or unmaintained projects
- Apps that are primarily something else (e.g. full project management suites)
- Duplicate entries
- Self-promotion without genuine merit
